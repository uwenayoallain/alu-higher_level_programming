# python-hello_world

This directory contains the following scripts:
- 0-run: Runs a Python file stored in \$PYFILE.
- 1-run_inline: Runs inline Python code stored in \$PYCODE.
- 2-print.py: Prints a specific sentence.
- 3-print_number.py: Prints an integer with a message.
- 4-print_float.py: Prints a float with 2-digit precision.
- 5-print_string.py: Prints a string three times and its first 9 characters.
- 6-concat.py: Prints a welcome message using string concatenation.
- 7-edges.py: Prints parts of a word (first 3 letters, last 2 letters, and middle letters).
- 8-concat_edges.py: Creates a sentence by concatenating variables.
- 9-easter_egg.py: Prints the Zen of Python.
